# Administration

*   `?config <paramètre> <valeur>`

    Configurer le serveur

    Alias: `?settings`

    **Arguments:**

    `<paramètre>` (requis): Paramètre à modifier

    `<valeur>`(requis): Valeur du paramètre
*   `?say <message>`

    Renvoie votre message

    Alias: `?repeat - ?rep`

    **Arguments:**

    \<message> (requis): Message a envoyer
